Which GPU?

Updated : 2017-09-26

Having access to GPU resources is essential for deep learning practitioners. Practically, there are two options : buy or rent. This means owning GPU(s) in some form or using cloud services.

​Siraj Raval reviews the following options in this video (2017-05-12) : AWS, Google Cloud and FloydHub.

His conclusions are :

  • AWS is the industry standard with the most options.
  • Google Cloud is cheaper and easier to use than AWS.
  • FloydHub is the easiest and best way to get started.
