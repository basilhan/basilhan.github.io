FloydHub Cookbook

Updated : 2017-10-03

This page describes concisely the minimal steps to run a job on FloydHub. The online documentation is comprehensive for most purposes and relevant references to it will be made as we go along.

There are two interfaces to work with : the browser and the python shell.

  • Login to FloydHub from the browser.
  • Open a shell.
    • Navigate to your code and data.
    • Execute :
      • > floyd login
    • Copy the authentication token from the browser and paste it.
    • Some control files will be generated.
  • ​Through the browser, create a project.
  • Through the shell, link your project to your code and data directory by executing :
    • > floyd init <project-name>
  • Some control files will be generated in the code and data directory.
  • Through the browser, create a new dataset.
  • Through the shell, upload the data :
    • > floyd data init <project-name>
    • > floyd data upload
  • ​Through the shell, run your python script or call up a Jupyter Notebook.
    • ​> floyd run ...arguments...
  • If your script generates output, take note to include the /output folder in the path.
  • When done,
    • Through the shell, logout :
      • > floyd logout
    • ​Through the browser, logout.
